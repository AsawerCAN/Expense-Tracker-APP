# Personal Finance Expense Tracker

A simple Python command-line application to track personal expenses.  
This project is built to practice Python, clean code structure, input validation, persistence, and basic testing.

---

## Features

- Add expenses with:
  - Date (YYYY-MM-DD)
  - Category (e.g. food, travel)
  - Description
  - Amount
- View all recorded expenses
- View total spending
- View total spending by category
- Persistent storage using a local JSON file
- Input validation to prevent crashes
- Unit tests using pytest

---

## Tech Stack

- Python 3
- Standard library only (no external runtime dependencies)
- pytest (for testing)
